CSS Selectors

I. The CSS element Selector

The CSS element selector selects the HTML element(s) you want to style.

The CSS element selector is written as follows:

Example:

p { text-align: right; color: blue; }

II. The CSS class Selector

The CSS class selector selects the HTML element(s) you want to style.

Example: 

        .class_name {
            padding:20px;
            text-align: right;
            color: yellow;
          }

Tag Example:

<p class="class_name"> This is Class Selector </p>

3. The CSS ID Selector

The CSS ID selector selects the HTML element(s) you want to style.

The CSS ID selector is written as follows:

Example:

#id_name { text-align: left; color: pink; padding:20px; }

3. The CSS grouping Selector

The CSS grouping selector selector selects the HTML element(s) you want to style.

The CSS grouping selector is written as follows:

Example:

* { text-align: left; color: pink; padding:20px; }

3. The CSS universal Selector

The CSS universal selector selects the HTML element(s) you want to style.

The CSS universal selector is written as follows:

Example:

h1, h2, h3 { text-align: left; color: purple; padding:20px; }
