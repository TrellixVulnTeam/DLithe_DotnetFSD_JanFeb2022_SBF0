Setting up of GIT in VS_CODE

How to download Git

1. Browse to the official Git website: https://git-scm.com/downloads

2. Click the download link for Windows and allow the download to complete.



3. After download Double-click the file to extract and launch the installer. and allow the file to Run

4. Then,Allow the app to make changes to your device by clicking Yes on the User Account Control dialog that opens. And click on goining Next until you get Install.

5. Open command promt and type the below command

git --version

6. You should see the version of git installed

7. on the Local Disk create a folder with somename.

8. Open Visual Studio code

9.In the visual studio click on "Open folder" and select the file in the step 7 where the file is located and select it.

10.In that folder the above image click on the "+file" icon and create file as index.html and write the some html code

11. Click Terminal --> New Terminal --> Change Powershell to Command Promt execute below commands

git config --global user.email "email_address"

git config --global user.name "github_username"

12. Click on "Source control button" and click on "initialize repository"

13. Stage changes by clicking "+" symbol

14. Click on Right symbol and type the first commit message

15. A browser window promts and click continue and authorize the github

16. Click on "Publish Branch"

For second commit

17. Follow Step 11 and Step 12.

18. Click on "Sync Changess"
